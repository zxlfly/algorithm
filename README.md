# 算法的学习记录
- **时间复杂度**
  - 常数阶O(1)
    - 没有循环等复杂结构
  - 对数阶O(logN)
    - 因素*2>n为止
  - 线性阶O(n)
    - 一个循环n表示次数
  - 线性对数阶O(nlogN)
    - 在对数阶O(logN)的基础上，外面加了循环次数为n
  - 平方阶O(n²)
    - 一个循环n嵌套一个循环n
    - 次数不一定要相等n*m也行
  - 立方阶O(n³)
    - 同上
  - K次方阶O(n^k)
    - 同上
  - 指数阶(2^n)
- **空间复杂度**
  - O(1)
    - 所需的临时空间不随变量的变化而变化
  - O(n)
    - 和n相关
